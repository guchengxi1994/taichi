# taichi(太极)

[EN](./README_en.md)



<img src="./md_images/taichi.gif" alt="taichi" width="20%" />

> 别盯着logo看，搞不好晃眼睛

一个包含一个或者多个组件的组件以及工具库，主要是开发过程中自己做的一些组件/功能或者一些挑战自我的实现😏。

## 1.安装

```
taichi:
    git:
        url: https://github.com/guchengxi1994/taichi
        ref: stable
```

> 如果想尝试新功能，可以使用 dev 分支或者 builder 分支
