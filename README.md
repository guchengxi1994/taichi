## 1.安装

> 前两天我想发布到 ```pub.dev```上的，但是又是身份验证又是要梯子，把我整不会了，所以现在只支持通过```git```这种方式安装。或者下载源码之后相对路径导入。

```
taichi:
    git:
        url: https://github.com/guchengxi1994/taichi
        ref: stable
```

> 如果想尝试新功能，可以使用 dev 分支或者 builder 分支
